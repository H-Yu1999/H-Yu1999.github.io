  • The paper of this project is still in progress.
  • Tools: Matlab, Simulink, Carsim

Optimal Demand Responsive Control for Buildings with Multi-Energy Systems

  • Design building resistance-capacitance model in BRCM toolbox based on government dataset. Design system model for demand response control. A detailed model for result verification and a simplified model for model based control will be created.

  • Implement and train the DRL models with popular frameworks like Tensorflow and PyTorch, improve the model with reducing the time used in training and increasing the accuracy. 3, lighten the model so that it can be run on a system that has limited computing resources.
